# Hero Image Requirements for New Blog Posts

This document specifies the hero images needed for the 4 new blog posts. Each post requires **ONE hero image only** (no inline images).

---

## Image Specifications (All Images)

**Dimensions:** 1200 x 630 pixels (ideal for social sharing - Facebook, Twitter, LinkedIn)  
**Format:** JPG or PNG  
**File Size:** Under 500KB (optimize for web)  
**Quality:** High resolution, professional appearance  
**Placement:** Place images in the specified folders with **exact filenames** listed below

## After Adding Images

1. Place each image in the correct folder with exact filename
2. Verify images are under 500KB (use online image compressor if needed: https://tinypng.com)
3. Test in development: `npm run dev`
4. Check that images appear correctly on blog listing page and individual blog posts
5. Deploy to production (Vercel auto-deploys from main branch)

---

## Image Optimization Tips

**To reduce file size without quality loss:**
- Use https://tinypng.com (drag and drop, download optimized)
- Or use Adobe Photoshop "Save for Web" (JPEG quality 75-85%)
- Or use ImageOptim (Mac) or FileOptimizer (Windows)

**Target:** 100-400KB per image (balance quality and load speed)
