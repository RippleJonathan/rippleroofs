## Tips for Adding Images

1. **Optimize file sizes:**
   - Keep under 500KB each for fast page load
   - Use tools like TinyJPG or Squoosh.app to compress

2. **Use descriptive filenames:**
   - Exactly as shown above (case-sensitive)
   - Helps with SEO

3. **Alt text is built in:**
   - The blog post already has alt text for screen readers
   - Just drop images in with correct filenames

4. **Test locally:**
   - After adding images, run `npm run dev`
   - Visit http://localhost:3000/blog/spring-hail-season-prep-austin
   - Verify all images load correctly
